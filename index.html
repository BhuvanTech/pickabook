Welcome to PickABook.ga

PickABook.ga is a P2P Book Dapp where people can buy or rent their favourite books using REVACoin (RUC).

It is built on the Ethereum Ropsten Test network.

In order to buy books you need metamask wallet and you need to import RUC using this hash - 0xE65b90cAad7B65651D747289378a5c086449E976
